# Manajemen Blog & Posting

Berikut adalah cara mengelola artikel blog Anda di WordPress.

### Membuat Artikel Baru
1. Cari menu **Posts** di sidebar kiri.
2. Klik **Add New**.
3. Masukkan judul dan isi konten.
4. Jangan lupa pilih **Category** dan masukkan **Featured Image**.
5. Klik **Publish**.

### Mengedit Artikel
Cukup klik judul artikel yang ada di daftar Posts, ubah, lalu klik **Update**.
