# Cara Membuat Artikel Baru

Dalam panduan ini, kita akan mempelajari cara membuat dan mempublikasikan artikel (blog post) baru di website Anda menggunakan sistem manajemen konten (CMS).

## Langkah 1: Masuk ke Dashboard
Pertama, pastikan Anda sudah login ke halaman admin / CMS website Anda. Biasanya dapat diakses melalui URL `/admin` atau URL khusus yang telah kami berikan.

## Langkah 2: Buka Menu Artikel
Di sidebar sebelah kiri, cari menu **"Posts"** atau **"Artikel"**, lalu klik **"Add New"** (Tambah Baru).

## Langkah 3: Mengisi Konten
Anda akan dihadapkan pada editor teks. Berikut adalah elemen yang wajib diisi:
1. **Judul:** Berikan judul yang menarik dan mengandung kata kunci (SEO friendly).
2. **Kategori:** Pilih satu atau beberapa kategori yang sesuai. Ini penting agar artikel terorganisir dengan baik.
3. **Featured Image (Gambar Utama):** Upload gambar berkualitas baik. Pastikan ukurannya di bawah 500KB agar website tetap cepat.
4. **Isi Artikel:** Tuliskan konten artikel Anda. Gunakan Heading (H2, H3) untuk memecah paragraf agar lebih mudah dibaca.

## Langkah 4: Publikasi
Setelah selesai, jangan lupa klik tombol **"Publish"** di pojok kanan atas. Atau klik **"Save Draft"** jika masih ingin disunting nanti.

*Catatan: Pastikan Anda selalu mengisi Meta Description di bagian bawah halaman untuk optimasi mesin pencari (SEO).*
