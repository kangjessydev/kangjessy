# Redesign Wisata Herbal Insani

## Latar Belakang & Tantangan

Wisata Herbal Insani menghadapi tantangan klasik dalam transformasi digital: website lama yang sudah tertinggal zaman (legacy) dengan antarmuka yang sangat padat oleh teks (information overload). Struktur konten yang fragmentasi mengharuskan pengelola membuat satu per satu halaman untuk setiap paket wisata baru, yang mana sangat tidak efektif secara administratif maupun pengalaman pengguna.

## Strategi Transformasi

Kami menerapkan pendekatan **Modern Nature Hub** dengan fokus pada tiga pilar utama:

1.  **Visual Storytelling**: Mengubah narasi panjang menjadi elemen visual interaktif, layout yang bernapas (whitespace), dan penggunaan warna hijau sebagai primer untuk memperkuat identitas brand.
2.  **UX Consolidation**: Membangun sistem **Tab Content** untuk paket wisata. Pengguna kini dapat membandingkan paket tanpa harus berpindah halaman, sementara admin dapat mengelola konten dengan lebih terpusat.
3.  **Scalable CMS Engine**: Implementasi **Custom Post Type (CPT)** khusus untuk Katalog Herbal, memungkinkan skalabilitas data tanaman obat yang terpisah dari konten reguler lainnya.

## Hasil Teknis

- **Dashboard Efisiensi**: Admin kini dapat merilis paket baru 2x lebih cepat.
- **Mobile-First Design**: Seluruh sistem tab dan katalog dioptimalkan untuk akses smartphone yang intuitif.
- **Clean Architecture**: Kode yang lebih modular dan seringan mungkin dengan performa Core Web Vitals yang optimal.
