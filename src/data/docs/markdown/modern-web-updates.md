# Update Konten Statis

Panduan untuk meng-update konten pada website berbasis modern stack (Non-CMS).

### Cara Update
1. Tim kami menyediakan file konfigurasi json/markdown sederhana.
2. Edit file tersebut untuk mengubah teks atau gambar.
3. Simpan perubahan Anda dan situs akan ter-update otomatis.

*Catatan: Sistem ini lebih aman dan lebih cepat daripada CMS tradisional.*
