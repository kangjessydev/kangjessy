Apa yang Terjadi?

Nur Alam Hotel, properti bintang 4 yang prestisius, mengalami serangan Japanese Keyword Hack — ribuan halaman berisi link judi online disuntikkan ke server secara diam-diam. Akibatnya, Google melabeli seluruh website sebagai berbahaya dan direct booking turun hingga 90% dalam waktu kurang dari 24 jam.

Tim IT internal sudah berusaha membersihkan, namun malware terus meregenerasi dirinya sendiri dalam hitungan menit. Ini adalah Polymorphic Backdoor yang ditanamkan jauh di file inti sistem, menyamar sebagai file lisensi plugin resmi.

"Sistem keamanan terbaik bukanlah yang paling mahal, tapi yang paling membosankan bagi peretas. Kami membuat server mereka 'mahal' untuk diretas."

Strategi & Eksekusi

Kami menerapkan pendekatan Defense in Depth — berlapis dari level plugin, server (Nginx config), hingga DNS (Cloudflare WAF). Prosesnya dimulai dengan audit forensik selama 2 jam sebelum menyentuh satu baris kode pun.

1. Forensic Audit

Scanning seluruh direktori menggunakan tools heuristik. Analisis 1.2 GB log server untuk melacak pola User-Agent aneh yang digunakan botnet saat menyuntikkan script.

2. Surgical Malware Removal

Pembersihan manual file-file terinfeksi tanpa merusak fungsionalitas website. Setiap file dibandingkan checksum-nya dengan repositori resmi WordPress.org untuk mendeteksi modifikasi sekecil 1 byte.

3. Hardening & WAF

Mematikan eksekusi PHP di folder /uploads dan /cache. Memasang Web Application Firewall via Cloudflare dengan aturan custom untuk memblokir botnet secara permanen.

Hasil & Dampak

Dalam waktu 4 jam kerja, server dinyatakan bersih total. Google mencabut label "Deceptive Site Ahead" dalam 2 minggu. Sebagai jaminan, kami memberikan garansi 12 bulan bebas infeksi ulang.
