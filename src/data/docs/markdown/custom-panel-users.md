# Kelola User & Role

Cara mengatur siapa saja yang bisa mengakses dashboard admin Anda.

### Daftar User
Buka menu **Settings > Users** untuk melihat daftar admin.

### Peran (Roles)
1. **Super Admin:** Akses penuh ke seluruh sistem.
2. **Editor:** Hanya bisa mengubah konten, tidak bisa mengubah pengaturan sistem.
3. **Viewer:** Hanya bisa melihat data tanpa bisa mengubahnya.

*Keamanan: Selalu pastikan tim Anda mengganti password secara berkala.*
