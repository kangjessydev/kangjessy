# Navigasi Panel Admin

Panduan untuk menggunakan dashboard kustom yang kami buat khusus untuk Anda.

### Sekilas Panel Admin
- **Beranda:** Ringkasan statistik & notifikasi.
- **Produk/Konten:** Kelola data utama website Anda.
- **Pengaturan:** Konfigurasi email & profil admin.

Sistem ini didesain agar mudah digunakan bahkan oleh orang awam teknis!
