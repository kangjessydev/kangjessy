# 📤 Panduan Upload & Aktivasi Template

Dokumentasi untuk instalasi tema/template baik melalui Dashboard maupun protokol FTP.

## 💻 Metode 1: Lewat Dashboard (Rekomendasi)
1. Masuk ke **Appearance** > **Themes**.
2. Klik tombol **Add New** di bagian atas.
3. Klik **Upload Theme** dan pilih file `.zip` template Anda.
4. Klik **Install Now** lalu **Activate**.



## 📁 Metode 2: Lewat FTP/File Manager
Gunakan metode ini jika ukuran file template terlalu besar:
1. Ekstrak file `.zip` template di komputer Anda.
2. Unggah folder hasil ekstrak ke direktori `/wp-content/themes/`.
3. Masuk ke Dashboard WordPress dan aktifkan tema tersebut.

## ⚠️ Hal Penting Setelah Upload
* **Install Plugins**: Cek notifikasi jika tema membutuhkan plugin pendukung (seperti Elementor atau Redux).
* **Import Demo**: Gunakan fitur *One-Click Demo Import* jika tersedia untuk mendapatkan tampilan seperti pratinjau.
